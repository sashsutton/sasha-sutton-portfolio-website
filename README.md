# Sasha Sutton, Portfolio

My personal website. Live at https://sashasutton.dev

Built with Next.js, TypeScript and Tailwind CSS.

## Run it locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Structure

- `app/` the pages (home, science, music, content)
- `components/` shared UI (nav, footer, sections, cards)
- `data/` all the site text, in English and French

The site has three sections, one for the AI and science work, one for the
music, and one for the videos. You can switch language and theme from the nav.
